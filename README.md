# Problem Solving Study

알고리즘 문제를 풀며 **코드와 접근 과정**을 기록하는 저장소입니다.

다음과 같은 문제를 위주로 정리합니다.

- ❌ **틀린 문제**
- ⌛ **풀이에 오래 걸린 문제**
- 🧐 **중요하거나 배울 점이 많은 문제**

## 📁 Repository Structure

**알고리즘 사이트 → 난이도 → 문제별 폴더** 구조로 정리되어 있습니다.

```
ps-study
├─ [알고리즘 사이트]
│  ├─ [난이도]
│  │  ├─ [세부 난이도]
│  │  │  └─ [문제번호].[문제이름]  # 문제별 폴더
│  │  │     ├─ [문제번호].cpp
│  │  │     └─ [문제이름].md
│  │  └─ ...
│  └─ ...
└─ README.md
```

### 예시

```
ps-study
├─ BOJ
│  ├─ Bronze
│  │  ├─ Bronze I
│  │  │  └─ 2869.달팽이는 올라가고 싶다
│  │  │     ├─ 2869.cpp
│  │  │     └─ 달팽이는 올라가고 싶다.md
│  │  ├─ Bronze II
│  │  └─ ...
│  ├─ Silver
│  └─ ...
├─ ...
└─ README.md
```

### 💼 문제별 폴더

각 문제는 하나의 폴더로 정리합니다.

문제 폴더에는 **풀이 코드**와 **풀이 정리 문서**가 함께 저장됩니다.

- 풀이 코드 (`문제번호.cpp`, `문제번호.py` ... )
- 풀이 정리 문서 (`문제이름.md`)
